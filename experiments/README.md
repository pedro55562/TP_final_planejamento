# Experiments

Scripts de cena e controle usados para testar planejadores em cenários completos.

- `rrt_se3/`: experimento do UAV/corpo rígido usando o RRT bidirecional em SE(3)
  embutido no UAIBot.

Esses scripts ficam fora do pacote `UAIbotPy` de propósito: eles são cenários de
uso, não implementação da biblioteca.
